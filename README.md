# AI Prompt Playground

A JavaScript fundamentals project featuring a text input for prompts, submit validation, and a live display of submitted prompts with history tracking. Built with vanilla HTML, CSS, and JavaScript as part of the AI Engineering Track — no frameworks, no AI model connected yet.

## Features

- Text area for entering a prompt
- Submit button with click and Ctrl+Enter support
- Input validation — blocks empty or whitespace-only prompts
- Displays the submitted prompt in a clean output card
- Tracks a history of the last 10 submitted prompts
- Clean, responsive UI styled with Flexbox

## Concepts Practiced

- Variables (`let`, `const`)
- Data types & operators
- Conditionals (`if`)
- Loops (`for`)
- Functions
- Arrays & Objects
- DOM manipulation (`createElement`, `textContent`, `classList`, event listeners)
- Fetch API — introductory stub (`sendPromptToAI`) for future AI integration

## Tech Stack

- HTML5
- CSS3 (Flexbox, custom properties)
- Vanilla JavaScript (ES6+)
